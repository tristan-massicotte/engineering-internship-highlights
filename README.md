# Engineering Internship Highlights

A bold, interactive portfolio presenting operational engineering internship projects through animated case studies and executive dashboards.

## Live site

[engineering-internship-highlights.medialexie.chatgpt.site](https://engineering-internship-highlights.medialexie.chatgpt.site)

## Featured projects

- MRP automation
- Inventory reduction
- Automatic RFQ-to-PO process
- Order follow-up automation
- Executive KPI dashboard

## Technology

- React 19
- TypeScript
- Vinext / Vite
- CSS and interactive SVG visualizations
- Cloudflare-compatible worker runtime

## Run locally

Requirements: Node.js 22.13 or newer.

```bash
npm ci
npm run dev
```

Then open the local URL shown in the terminal.

## Validation

```bash
npm run lint
npm run build
```

## Data and confidentiality

All organization names, supplier names, product identifiers, and operational simulations shown in this portfolio are fictional placeholders. Percentages identified as actual outcomes are retained where stated; confidential monetary values are fictionalized.
